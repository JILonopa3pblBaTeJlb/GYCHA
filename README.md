📻 Radio GYCHA Core 
Профессиональный программный комплекс для организации круглосуточного потокового вещания (RTMP) с динамическим ASCII-графическим интерфейсом, автоматизированным сбором контента из Telegram и интеллектуальным управлением эфирными блоками.
Система оптимизирована для работы на высокопроизводительных серверах (тестировалось на AMD Ryzen 9) под управлением Python 3.10.12.
🚀 Основные возможности
* Динамический UI: Генерация текстового интерфейса в реальном времени (курсы валют, погода, новости, системные ресурсы, расписание).
* Умная ротация: Генерация плейлистов с защитой от повторов и учетом жанровых сеток.
* VHS Mode: Автоматическое переключение на трансляцию фильмов в режиме Picture-in-Picture (PiP) в заданные дни и часы.
* Content Auto-Ingection: Автоматический парсинг и верификация медиа-файлов из привязанных Telegram-каналов.
* Отказоустойчивость: Автоматический перезапуск процессов FFmpeg при крашах или зависаниях с сохранением позиции воспроизведения.
* Интерактивность: Прием сообщений от слушателей через Telegram-бота и вывод их в прямой эфир.
📂 Обзор модулей системы
🧠 Ядро и управление процессами
* air_1h.py: Главный диспетчер (Orchestrator). Управляет жизненным циклом эфира, переключает режимы (Радио/VHS) и следит за таймингами.
* process_manager.py: Высокоуровневая обертка над процессами. Отслеживает "зависание" потока (отсутствие логов), обрабатывает аварийное завершение и управляет освобождением RTMP-сокетов.
* air_supply.py: Набор утилит для расчета времени следующего события и гарантированной остановки подпроцессов.
* config_loader.py: Универсальный загрузчик config.ini с поддержкой автоматического приведения типов и горячей перезагрузки параметров без остановки системы.
📺 Генерация UI (Graphics Engine)
* status.py: Сердце визуализации. Собирает данные из всех модулей и формирует финальный текстовый буфер fakelogs.txt, который FFmpeg накладывает на видео.
* clock.py: Генератор ASCII-часов с кастомным шрифтом.
* diag.py: Модуль диагностики сетевых задержек (MTR) и нагрузки системы (vmstat) для вывода в эфир.
* uvb.py: Генератор кодовых сообщений в стиле номерных радиостанций (УВБ-76) для заполнения пауз и создания атмосферы.
* res.py: Мониторинг системных ресурсов: CPU, RAM и дискового пространства.
🎵 Управление контентом
* 1h_pl.py: Генератор часовых плейлистов. Формирует структуру часа (джинглы, реклама, треки) на основе hour.txt и program.txt.
* content.py: Автоматизированный загрузчик. Сканирует Telegram-каналы, проверяет файлы через ffprobe на целостность и соблюдение лимитов длительности.
* recorder.py: Автоматически записывает прошедший эфирный час и выкладывает его в архивный Telegram-канал с метаданными.
🌐 Внешние интеграции
* weather.py: Пакетный сбор данных о погоде для списка городов через Open-Meteo API.
* finance.py: Получение актуальных курсов криптовалют и фиатных пар через CoinGecko.
* rss.py: Агрегатор новостей из Google News.
* messenger.py: Telegram-бот для слушателей. Поддерживает модерацию через черный список слов и выводит сообщения в "бегущую строку".
* watchdog.py: Монитор Twitch-трансляции. Если стрим перезапускается, бот автоматически обновляет ссылку в закрепленном сообщении канала.
🎥 Работа с видео (FFmpeg)
* ffmpeg_runner.py: Конструктор сложных команд FFmpeg для стандартного вещания с наложением оверлеев.
* vhs_runner.py: Специализированные команды для режима киносеанса с поддержкой возобновления с места разрыва.
